# OpenClaw CLI-Anything Skill

[![OpenClaw](https://img.shields.io/badge/OpenClaw-Skill-blue)](https://openclaw.ai)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

An OpenClaw skill for building CLI-Anything harnesses that bring command-line interfaces to GUI applications.

## What is CLI-Anything?

CLI-Anything is a methodology for creating command-line interfaces for applications that don't have one. It generates Python harnesses that can:

- 🚀 Launch GUI applications programmatically
- ⌨️ Send keystrokes and mouse actions
- 📷 Read screen output via OCR
- 🤖 Automate complex workflows

## Installation

### Via ClawHub (Recommended)

```bash
openclaw skill install cli-anything
```

### Manual Installation

1. Clone this repository:
```bash
git clone https://github.com/liwei626/openclaw-skill-cli-anything.git
```

2. Copy to your OpenClaw skills directory:
```bash
cp -r openclaw-skill-cli-anything ~/.openclaw/skills/cli-anything
```

## Usage

Once installed, simply ask OpenClaw to help you create a CLI for any application:

```
"帮我为 Photoshop 创建一个 CLI 工具"
"Create a CLI harness for my custom GUI app"
"Build an automation script for Excel operations"
```

## Features

- **Smart Analysis**: Automatically identifies UI elements and interaction patterns
- **OCR Integration**: Reads screen content for feedback and validation
- **Error Handling**: Robust error detection and recovery mechanisms
- **Cross-Platform**: Works on Windows, macOS, and Linux

## Example Output

The skill generates a complete Python harness:

```python
#!/usr/bin/env python3
"""
CLI harness for MyGUIApp
Generated by OpenClaw CLI-Anything Skill
"""

import argparse
import subprocess
import time
import pyautogui
from pynput.keyboard import Controller, Key

def main():
    parser = argparse.ArgumentParser(description='CLI for MyGUIApp')
    parser.add_argument('--open', help='Open file')
    parser.add_argument('--save', help='Save to file')
    parser.add_argument('--action', choices=['process', 'export', 'print'])
    
    args = parser.parse_args()
    
    # Launch application
    print("Launching MyGUIApp...")
    proc = subprocess.Popen(['MyGUIApp'])
    time.sleep(3)  # Wait for app to load
    
    # Perform actions based on arguments
    if args.open:
        pyautogui.hotkey('ctrl', 'o')
        time.sleep(0.5)
        pyautogui.typewrite(args.open)
        pyautogui.press('enter')
    
    # ... more automation logic
    
    proc.terminate()

if __name__ == '__main__':
    main()
```
